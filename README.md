# 《A-shares：中国A股》

本人在2017年2月6日开始在A股有了第一笔交易，持仓至今，入行时间不长，累计收益为正。在股市连滚带爬数年，对股票的认识仍然不足。创建本项目希望自己一边可以做一名理智合格的持股人，一边希望自己股票的账户资产早日突破百万。以过去看现在，用现在看未来。作者：[0e0w](https://github.com/0e0w)

本项目创建于2021年9月1日，最近的一次更新时间为2022年2月1日。
